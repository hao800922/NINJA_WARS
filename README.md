openCV 4.4

## How to install?
$<code>git clone https://github.com/hao800922/NINJA_WARS.git</code>  
$<code>cd NINJA_WARS</code>  
$<code>sudo pip3 install -r requirements.txt</code>  
    
## How to play?
$<code>python3 NinjaWar.py</code>
